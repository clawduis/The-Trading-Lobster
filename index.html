The Biology

Why a lobster?

The lobster's stomatogastric ganglion (STG) is one of the most studied neural circuits in all of biology. It's been under investigation since the 1970s. It has only ~30 neurons but generates complex, robust rhythmic firing patterns — making it ideal for recording.

The STG controls the lobster's stomach muscles. When the lobster is in an active, excited state, neurons fire in rapid bursts at 15–20 Hz. When inhibited or resting, firing drops to 2–5 Hz. These two states map cleanly to LONG and SHORT.

The ganglion is dissected from the animal and kept alive ex vivo in cold oxygenated saline for up to 12 hours — plenty of time to trade.

Risk Management

The lobster doesn't know what Bitcoin is.

That's why we built a full risk engine around it.

🛑 Daily Loss Halt
5%
If the account drops more than 5% in a single day, all trading halts automatically until manual reset.
📉 Stop Loss
2.5%
Each position is automatically closed if unrealized loss exceeds 2.5% of entry price.
📈 Take Profit
5%
Positions are closed when unrealized gain hits 5%. The lobster doesn't get greedy.
⏱️ Cooldown
5 min
Minimum 5-minute cooldown between consecutive trades. Prevents rapid-fire signal noise from overtrading.
💰 Position Cap
$500
Hard maximum on open notional. Default trade size is $50. Configurable in config.yaml.
🔒 Signal Confidence
60%
Only trades when classifier confidence is above 60%. Ambiguous states result in HOLD — no action taken.

Get Started

Run it in 3 commands

No hardware required to test. The mock reader generates synthetic neural signals to exercise the full pipeline.

bash — The Trading Lobster
# Clone and install
$ git clone https://github.com/clawduis/The-Trading-Lobster.git
$ pip install -r requirements.txt
 
# Copy and fill in your Hyperliquid private key + wallet address
$ cp .env.example .env
 
# Test fully offline — synthetic signal, no trades
$ python main.py --mock --dry-run
 
# Run on testnet with mock signal
$ python main.py --mock
 
# Go live — real lobster, real trades
$ python main.py
